# 排版规范手册

本项目的 LaTeX 样式通过 `preamble.tex` 定义，编译入口为 `compile.ps1`（Windows）或 `compile.sh`（Unix）。

**这份手册只面向 AI 代理**，说明如何将 PDF/OCR 文本转换为符合本项目规范的高质量 LaTeX 源码。

---

## 1. 核心排版环境

### 1.1 文件结构

```latex
% 文件头部
\heading{课程名-年份学期-类型}
\noteinfo{来源说明，如 OCR 精修}

% 大题标题（放在 question 环境外）
\textbf{一、题型名称（共 XX 分）}

% 题目
\begin{question}[分数]
题面文字（环境自动编号，不要写 1. 或 2.）
\end{question}
```

### 1.2 题目环境：`question`

| 用途 | 写法 |
|------|------|
| 普通题目 | `\begin{question}[2 分]` |
| 无分数 | `\begin{question}` |
| 结束 | `\end{question}` |

**禁止**在 `\begin{question}` 后写 `1.\\`、`1.`、`1、` 等手动编号。环境已自增编号。

大题标题（`\textbf{一、选择题（共 20 分）}`）放在 `\begin{question}` **之外**，不与任何题目环境嵌套。

### 1.3 选择题：`choiceoptions`

```latex
\begin{choiceoptions}[2]   % 参数=列数，默认2
  \item 选项A文字
  \item 选项B文字
  \item 选项C文字
  \item 选项D文字
\end{choiceoptions}
```

列数建议：
- 选项短（1-2个词）→ `[4]` 或 `[2]`
- 选项一般 → `[2]`
- 选项很长（完整句子）→ `[1]`

**禁止**：`A. xxx`、`B. xxx`、`A．xxx`、`B．xxx`、`(A) xxx` 等手动编号。

### 1.4 填空题：`\blank`

```latex
\blank          % 默认宽度 1.5cm
\blank[3cm]     % 指定宽度
```

**禁止**：`_____`、`____`、`___`、`______` 等原始下划线。

### 1.5 解答环境：`solution`

```latex
\begin{solution}
完整推导过程，用 \boxed{} 标出最终答案。
\end{solution}
```

`solution` 中的内容在编译答案版本时显示（由 `\ifShowSolutions` 控制）。

### 1.6 代码块

```latex
% 通用代码（C++/Java/伪代码）—— 等宽字体，浅灰色框
\begin{ttbox}
int sum = 0;
for (int i = 0; i < n; i++) {
    sum += i;
}
\end{ttbox}

% 算法描述 —— 蓝色框
\begin{codebox}
def sort(arr):
    return sorted(arr)
\end{codebox}

% Python 代码 —— 带行号，verbatim 模式
\begin{pycode}
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b
\end{pycode}
```

**注意**：
- `\begin{ttbox}` 内部用 `\ttfamily`，`_`、`&`、`{`、`}` 需转义
- `\begin{pycode}` 和 `\begin{codebox}` 是 verbatim 模式，无需转义
- 代码中有 `_` 等特殊字符时优先用 `\begin{codebox}`

### 1.7 列表：`enumerate`

```latex
% 一级：(1), (2), (3)
\begin{enumerate}
  \item 第一项
  \item 第二项
\end{enumerate}

% 二级：(a), (b), (c)
\begin{enumerate}
  \item 主项
  \begin{enumerate}
    \item 子项
  \end{enumerate}
\end{enumerate}
```

样式在 `preamble.tex` 中全局设置，不需手动指定。样式层级为 `(1)` → `(a)` → `(i)` → `(A)`。

### 1.8 答案标记

```latex
\ansmath{E=mc^2}       % 行内公式答案
\anstext{正确答案}      % 文字答案
```

## 2. 解答/答案书写规范

**总原则**：尽可能详细，保留关键运算步骤，不要跳步。考试参考答案的标准应该是"每一步代数推导都写出来"。

### 2.1 推导完整性

```latex
% ❌ 太简略（不合格）
E_n^{(1)} = 2α/a·sin²(nπ/2)

% ✅ 完整（合格）
一阶能量修正为
\[
E_n^{(1)} = \langle n|H'|n\rangle
= \int_0^a \psi_n^*(x)\alpha\delta(x-a/2)\psi_n(x)\,\dd x
= \alpha|\psi_n(a/2)|^2.
\]
代入波函数 $\psi_n(x)=\sqrt{2/a}\sin(n\pi x/a)$：
\[
|\psi_n(a/2)|^2 = \frac{2}{a}\sin^2\frac{n\pi}{2}.
\]
因此
\[
\boxed{E_n^{(1)} = \frac{2\alpha}{a}\sin^2\frac{n\pi}{2}.}
\]
当 $n$ 为偶数时 $\sin(n\pi/2)=0$，修正为零；$n$ 为奇数时修正为 $2\alpha/a$。
```

### 2.2 不同题型的展示内容

| 题型 | 必须展示的中间步骤 |
|------|-------------------|
| 时间复杂度 | 递推式 → 展开 → 求和 → 取大 O |
| 排序 | 每趟排序后的序列状态 |
| 散列表 | 每个关键字地址计算、冲突解决过程 |
| 树操作 | 插入/删除的每步树形变化 |
| 图算法 | 每步访问的顶点/边序列 |
| 变分法 | 归一化 → ⟨T⟩ → ⟨V⟩ → E(b) → 求极值 → 答案 |
| WKB | 量子化条件 → 变量代换 → 积分 → 化简 |
| 微扰论 | 矩阵元积分 → 求和式展开 → 一阶/二阶修正 |
| 动态规划 | 状态定义 → 转移方程 → 填表过程 |
| 数学证明 | 归纳假设 → 归纳步推导 → 结论 |

### 2.3 最终答案标记

所有最终答案用 `\boxed{}` 包裹：

```latex
\[
\boxed{ASL_{\text{succ}} = \frac{12}{7}}
\]

\boxed{\text{A}}
```

## 3. 图表处理

### 3.1 能用 TikZ 画的图优先用 TikZ

**二叉树**：
```latex
\begin{tikzpicture}[level distance=9mm,
  every node/.style={draw,circle,minimum size=6mm,inner sep=1pt},
  level 1/.style={sibling distance=26mm}]
\node {根}
  child {node {左子}}
  child {node {右子}};
\end{tikzpicture}
```

**红黑树**（区分红/黑结点）：
```latex
\begin{tikzpicture}[
  blacknode/.style={draw,rectangle,fill=black,text=white,minimum size=6mm},
  rednode/.style={draw,circle,red,text=red,minimum size=6mm}]
\node[blacknode] {根}
  child {node[rednode] {红结点}};
\end{tikzpicture}
```

**B 树**：
```latex
\tikzset{bt/.style={draw,rectangle,minimum height=6mm,inner sep=3pt}}
\node[bt] (r) at (x,y) {key1\quad key2};
```

**有向图**：
```latex
\begin{tikzpicture}[>=stealth]
  \node[draw,circle] (v1) at (0,0) {$V_1$};
  \node[draw,circle] (v2) at (2,0) {$V_2$};
  \draw[->,thick] (v1) -- (v2);
\end{tikzpicture}
```

**TikZ 图始终用 `\begin{center}` 居中**。

### 3.2 插入图片

仅当 TikZ 无法高效复刻时（复杂几何、照片）使用：
```latex
\pyfig[0.6\linewidth]{path/to/image.png}{图注文字}
```

### 3.3 解答中嵌入 TikZ

答案中的示意图直接放在 `\begin{solution}` 内：
```latex
\begin{solution}
插入后的树为：
\begin{center}
\begin{tikzpicture}[...]
  ...
\end{tikzpicture}
\end{center}
\end{solution}
```

## 4. 数学约定

```latex
\ii     % 虚数单位 i（正体）
\ee     % 自然常数 e（正体）
\dd     % 微分算符 d（正体）
```

常用物理符号（来自 `physics` 宏包）：`\bra{}` `\ket{}` `\braket{}` `\grad` `\div` `\curl`。

## 5. 从 PDF 到 TeX 的工作流

```
PDF/图片
  ↓ ① OCR（PaddleOCR / Tesseract / Mathpix）
  ↓ ② 人工校对（重点：公式、上下标、特殊符号）
  ↓ ③ 按本规范转为 LaTeX
  ↓ ④ 放入对应目录 exam/<课程名>/ 或 homework/<课程名>/
  ↓ ⑤ compile.ps1 / compile.sh 编译验证
```

### OCR 后必校项

- [ ] 数学符号：α β ∑ ∫ √ 是否识别正确
- [ ] 上下标：`x²`→`x^2`，`x₂`→`x_2`
- [ ] 花括号、括号匹配
- [ ] 代码中的 `_` `&` `{` `}` 是否被吞
- [ ] 英文变量大小写

## 6. 绝对禁止清单

| ❌ 错误写法 | ✅ 正确写法 | 原因 |
|------------|------------|------|
| `\begin{question}` 后写 `1.\\` | 直接写题面 | 环境自动编号 |
| `A. xxx  B. xxx` | `\begin{choiceoptions}` | 统一选项排版 |
| `_____` | `\blank` | 统一填空格式 |
| 大题标题包在 question 内 | 放在 question 外 | 结构分层清晰 |
| 行末残留 `\\` | 删除 | LaTeX 自动换行 |
| 手写 `(1)(2)` 列表 | `\begin{enumerate}` | 统一列表样式 |
| 推导只写结果 | 写完整代数过程 | 考试答案需要评分点 |
| 代码用普通文本 | `\begin{ttbox}` / `\begin{codebox}` | 等宽字体 |
| 代码 tab 缩进 | 空格缩进 | 跨编辑器兼容 |
| OCR 文本描述树 | TikZ 画树 | 可视化且可维护 |
| 方案中用具体课程名举例 | 用抽象示例 | 通用模板 |

## 7. 编译验证

转换后必须运行编译并确认无错误：

```bash
.\compile.ps1   # Windows
bash compile.sh # Unix
```

常见错误及修复：

| 错误 | 原因 | 修复 |
|------|------|------|
| `Undefined control sequence` | 命令拼写错误或未定义 | 检查 `\` 命令名 |
| `Missing \endgroup` | 环境未配对 | 检查 `\begin`/`\end` 配对 |
| `Missing $ inserted` | 数学模式未正确开闭 | 检查 `$` 或 `\[` `\]` 配对 |
| `Extra } or forgotten $` | 花括号不匹配 | 逐层检查 `{` `}` |
| Package 编码错误 | 不可见 Unicode 字符 | 清理 OCR 残留字符 |

## 8. 文件命名规则

```
exam/<课程名>/<课程名>-<年份><学期>-<类型>.tex
homework/<课程名>/<第N章>-<章节名>.tex
```

- 课程名中的括号用全角：`高等数学A（II）`
- 年份两位：`14` `15` `22` 等
- 学期：`春` `秋`
- 类型：`期中` `期末` `模拟题N` `免修考`
