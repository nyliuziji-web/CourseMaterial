# 协作者手册

## 可以贡献什么

- 往年试卷 → `exam/<课程名>/`，记得写答案（可以直接让 ai 生成）
- 课后作业 → `homework/<课程名>/`
- 课程讲义与资料 → `lecture/<课程名>/`（目前为空目录，欢迎大家补充）
- 命名示例：`高等数学A（II）-25春-期末.tex`、`第3次作业.tex`

## 排版规则

- 每个 .tex 文件开头写 `\heading{课程名-说明}`，用 `question` 环境包裹题目，`solution` 环境包裹解答
- 配图用 `\pyfig[宽度]{路径}{说明}`，图放在对应课程的 `figures/` 文件夹
- 不要写 `\noteinfo`、`\bigskip`、`\ifShowSolutions`。不用管 `\begin{document}`，每个 .tex 只是一段内容

## 提交

```bash
git add -A
git commit -m "feat: 添加了……"
git push
```

如果被拒，先 `git pull --rebase` 再 push。更详细的规范见旧版 `contributer_manual.md`。
