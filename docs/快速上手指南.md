# 快速上手指南

## 只想看 PDF

1. 打开 <https://github.com/claudemt/CourseMaterial>
2. 点绿色的 **Code** → **Download ZIP**
3. 解压后进入 `example/` 文件夹，打开 PDF 即可

## 想要自己挑着编译

需要电脑装了 TeX Live（完整版）。

1. 下载/克隆本项目
2. 打开终端进入项目根目录
   - **Windows**：运行 `.\compile.ps1`
   - **macOS / Linux**：运行 `./compile.sh`
3. 跟着菜单选内容，脚本自动生成 PDF 到 `build/` 文件夹
