\begin{solution}
一、选择与填空：
\begin{enumerate}[leftmargin=2em]
  \item 选 $O(2^n)$。每次递归产生两个规模为 $n-1$ 的子问题，递归树叶结点数为 $2^n$。
  \item A、C、D 的赋值次序均能正确完成在双循环链表结点 $p$ 后插入 $s$；B 错在过早令 \texttt{\detokenize{p.next=s}}，随后 \texttt{\detokenize{s.next=p.next}} 会使 \texttt{\detokenize{s.next}} 指向自身。
  \item 容量至少为 3。过程为：压入 $e_1,e_2$ 后弹出 $e_2$；压入 $e_3,e_4$ 后弹出 $e_4,e_3$；压入 $e_5,e_6$ 后弹出 $e_6,e_5,e_1$，最大栈深为 3。
  \item 按通常约定 \texttt{\detokenize{front}} 指向队首、\texttt{\detokenize{rear}} 指向下一个可插入位置，元素个数为 $(rear-front+40)\bmod40$，故分别为 $7$ 和 $33$。
  \item 若只计非空子串，为 $n(n+1)/2$；若按教材定义把空串也计入，则为 $n(n+1)/2+1$。
  \item 度为 2 的结点最多为 $\lfloor(n-1)/2\rfloor$。
  \item 若堆数组采用 0 下标，最大元素只能出现在叶结点位置，范围为 $25$ 到 $49$；若采用 1 下标，则为 $26$ 到 $50$。
  \item 仅给出一个 Huffman 码字 \texttt{\detokenize{1101}} 不能唯一确定按频率加权的平均编码长度；若补充约定 5 个字符等概率，且码长集合为 $1,2,3,4,4$，则平均编码长度为 $14/5$。
  \item $I$ 的父结点是 $C$。由先根序 $GABCDIJEFH$ 与后根序 $BADIJCFEHG$ 可知 $G$ 的孩子依次为 $A,C,E,H$，而 $C$ 的孩子为 $D,I,J$。
  \item 补全题意为“从倒数第二层任取一个结点，该结点为叶结点的概率”。完全三叉树前 5 层共有 $1+3+9+27+81=121$ 个结点，144 个结点说明第 6 层有 23 个结点，占据第 5 层前 $\lceil23/3\rceil=8$ 个父结点，因此第 5 层叶结点数为 $81-8=73$，概率为 $73/81$。
\end{enumerate}

二、辨析与简答：
\begin{enumerate}[leftmargin=2em]
  \item 用 Floyd 快慢指针。先令慢指针每次走一步、快指针每次走两步，二者相遇则进入环；从相遇点继续沿环走一圈并计数即可得到循环链表元素个数。空间复杂度 $O(1)$。
  \item 扫描操作串，用计数器记录当前栈深：遇到 I 加 1，遇到 O 减 1。任意前缀中 O 的数量不得超过 I，即栈深不能为负；扫描结束时 I 与 O 数量相等，即栈深为 0。两条件同时满足则合法。
  \item 初始数组 $\{10,5,12,3,2,1,8\}$ 经筛选建最大堆得 $\{12,5,10,3,2,1,8\}$，主要交换 $10\leftrightarrow12$。依次插入 $6,7,14$ 后，插入 14 的交换为 $14\leftrightarrow2$、$14\leftrightarrow7$、$14\leftrightarrow12$，得到 $\{14,12,10,6,7,1,8,3,5,2\}$。随后连续 3 次 \texttt{\detokenize{deleteMax}} 后堆为 $\{8,7,5,6,2,1,3\}$；第 3 次删除中的下滤交换为 $3\leftrightarrow8$、$3\leftrightarrow5$。
  \item 按题给重量权衡规则并在查找时路径压缩，所有 $0\sim15$ 最终合并为一个等价类，代表元可取 $0$。完全压缩后的等价类树可画成以 $0$ 为根，其余结点均为 $0$ 的孩子。
\end{enumerate}

三、算法填空：
\begin{enumerate}[leftmargin=2em]
  \item 一个排列可由一个栈排成升序，当且仅当不存在 $i<j<k$ 且 $sequence[k]<sequence[i]<sequence[j]$ 的 231 模式。填空为：

    \item \texttt{\detokenize{cur <= len}}；

  \end{enumerate}
  \item 带右链先根序转带度数后根序的填空：

    \item \texttt{\detokenize{PostTree[count].info = node.info}}；

    \item \texttt{\detokenize{node.rlink is not None}}；
    \item \texttt{\detokenize{return 1}}。
  \end{enumerate}

四、算法设计与实现：
\begin{enumerate}[leftmargin=2em]
  \item 可用后缀数组。构造所有后缀及其起始下标，按字典序排序；最长重复子串一定是相邻后缀的最长公共前缀之一。依次计算相邻后缀 LCP，取最长者及较小起始下标即可。朴素实现时间 $O(n^2\log n)$，若用线性 LCP 算法可降为 $O(n\log n)$ 或更优。
  \item 树高递归算法：若结点为空，返回 $-1$；若为叶子，返回 $0$；否则返回所有孩子树高最大值加 1。若按“层数”定义树高，则空树为 0，叶结点为 1，相应整体加 1 即可。
\end{enumerate}

五、证明题：
\begin{enumerate}[leftmargin=2em]
  \item 随机插入得到的 BST 与快速排序递归树同分布。设 $E[I_N]$ 为内部路径长度期望，根的左子树规模 $i$ 在 $0\sim N-1$ 上等可能，因此
  \[
  E[I_N]=N-1+\frac1N\sum_{i=0}^{N-1}(E[I_i]+E[I_{N-1-i}]),
  \]
  解得 $E[I_N]=O(N\log N)$。
  \item 树的孩子兄弟表示中，树的“第一个孩子”成为对应二叉树的左孩子，“右兄弟”成为右孩子。树的先根遍历先访问根，再依次访问各子树；对应二叉树的前序遍历也是先访问根，再沿左孩子进入第一个子树，并沿右孩子依次访问兄弟子树。因此两者访问序列相同。
\end{enumerate}
\end{solution}
